# 🍽️ Mamays Puno - Sabores que Unen

**Mamays** es una plataforma Marketplace diseñada para conectar a comensales con "Mamays" (cocineras locales) de la ciudad de Puno. La misión es rescatar el sabor de la comida casera y tradicional, permitiendo que cualquier persona disfrute de un menú saludable y auténtico hecho con amor.

## 🚀 Características Principales

- **Sistema de Roles Dual:** Registro para **Compradores** (buscan comida casera) y **Cocineros** (vendedores que comparten su sazón).
- **Perfiles Personalizados:** Gestión completa de perfiles incluyendo preferencias alimenticias, alergias y datos de contacto.
- **Vendedores Certificados:** Los cocineros pueden subir fotos de su cocina y comedor para generar confianza.
- **Explorador de Platos:** Feed dinámico con filtros por categorías (Sopas, Segundos, Postres, etc.).
- **Detalle e Historia:** Cada plato muestra su origen, ingredientes, fotos del local y contacto directo vía WhatsApp para pedidos.
- **Persistencia en la Nube:** Integración total con MongoDB Atlas para garantizar la disponibilidad de los datos.

## 🛠️ Stack Tecnológico

- **Frontend:** React.js, Material UI, SASS.
- **Backend:** Node.js, Express.js.
- **Base de Datos:** MongoDB Atlas (Cloud).
- **Infraestructura:** Docker, Kubernetes (Minikube).
- **Escritorio:** Electron (Versión de escritorio nativa).
- **Despliegue:** Vercel (Frontend).

## 📦 Instalación y Uso Local

### Requisitos:
- Node.js (v20 o superior)
- Docker y Docker Compose

### Pasos:
1. Clonar el repositorio: `git clone https://github.com/Arnold123Diego/mamays_1g.git`
2. Instalar dependencias: `cd Zomato-Clone && npm install`
3. Levantar servicios: `sudo docker-compose up -d`
4. Iniciar la web: `npm start`
5. Iniciar versión escritorio: `npm run electron-dev`
